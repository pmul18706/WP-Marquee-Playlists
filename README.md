# WP Marquee Playlists

Create scheduled image playlists and display them as a marquee scroll or static gallery via a public link or shortcode. Built for graphics playback use-cases (e.g., billboards, lobby displays, digital signage).

## Features
- Upload media and add items to playlists
- Schedule items with start/end date-time windows
- Display playlists as marquee scroll or static gallery
- Public link per playlist (and shortcode support if enabled in the plugin)

## Installation
1. Copy this folder to: `wp-content/plugins/wp-marquee-playlists/`
2. In WordPress admin: **Plugins â†’ Installed Plugins**
3. Activate **WP Marquee Playlists**

## Usage (high level)
- Create a playlist in the plugin admin
- Add media items and set scheduling
- Use the generated public link to display the playlist
- If supported in your build, use the shortcode shown in the admin UI

## Versioning / Releases
- Update the plugin header `Version:` field
- Update `CHANGELOG.md`
- Tag releases as `vX.Y.Z`
